Learn Git Branching - Tutorial/activity

This website was quite useful shake off some of the rust I had with Git. I have used Git in the past, but it has been quite a while since I've used it truthfully, so this resource that Dr. Bart left in the discord helped me out quite a bit in refamiliarizing myself with Git version controlling again. I then continued on with the website activity, as it also included some advanced exercises towards the end. I will admit the advanced topics were challenging, but I learned about some new aspects of Git, such as Git cherry picking. With the help of some research outside of the website to help me out, I feel as if I learned a lot about git after using this website.

Typescript Playground - Tutorial/activity

This is one of the activities that was shared on the portfolio assignment page on Canvas. I played around with some of the activities on this page, and got my feet wet with typescript. I particularly thought the "any" and "unknown" primative types were interesting. I never expected such flexibiltiy with a programming language before, basically allowing to account for situations in which one does not know the type, for example. I understood generally how types and interfaces worked from my work with other languages, but the sections that this site had on them worked as a good refresher and helped me see how they work specifically within typescript. The typescript playground did a good job of introducing me to the language and refreshing some concepts for me.
